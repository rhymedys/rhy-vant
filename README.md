## Features

* 95% Unit test coverage
* Extensive documentation and demos
* Support [babel-plugin-import](https://github.com/ant-design/babel-plugin-import)
* Support Custom Theme
* Support i18n
* Support TS
* Support SSR

## Install

```bash
# Using npm
npm i vant -S

# Using yarn
yarn add vant
```
